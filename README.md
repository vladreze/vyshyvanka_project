# Графічний редактор орнаментів «Редактор Вишиванки»
Спеціалізований десктопний додаток, розроблений мовою Java з використанням фреймворку JavaFX та системи збірки Maven. Програма призначена для проектування геометричних візерунків української народної вишивки хрестиком у стилі піксель-арту.

## Функціональні можливості програми

- Інтерактивне робоче поле:
Візуальна сітка (Grid Canvas) з можливістю зафарбовування клітинок мишею, вибором кольору ниток через вбудовану палітру кольорів та інструментом для видалення помилкових стібків.
  
- Система дзеркального відображення в реальному часі: Автоматичне дублювання стібків за обраними осями для швидкого створення симетричних циклічних візерунків:
  Горизонтальна симетрія (віддзеркалення зліва направо).
  Вертикальна симетрія (віддзеркалення зверху вниз).
  Радіальна симетрія (одночасна активація обох режимів для генерації складних орнаментів та зірок).
  
- Імпорт та експорт даних:
Збереження поточного стану малюнка у файл даних проєкту для можливості подальшого завантаження та редагування.
Експорт готового візерунка у форматі растрового зображення високої якості (PNG) без ліній робочої сітки для зручного друку на папері чи тканині.

- Довідка користувача:
  Вбудоване модальне вікно з інструкціями та демонстраційними матеріалами для швидкого ознайомлення користувача з інтерфейсом додатка.

## Технологічний стек та архітектура

Мова програмування: Java (JDK 17 або новіша)
Графічна бібліотека: JavaFX
Система збірки: Maven
